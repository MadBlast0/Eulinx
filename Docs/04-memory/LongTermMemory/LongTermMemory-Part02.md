# Review

Eulinx should support a review queue before promoting AI-created memory to long-term memory.

# Editing

Users should be able to:

- edit memory
- pin memory
- archive memory
- delete memory
- mark memory incorrect

# Forgetting

Forgetting is required when memory becomes stale, sensitive, incorrect, or user-rejected.
