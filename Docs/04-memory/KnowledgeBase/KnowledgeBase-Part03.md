# Retrieval

Workers should receive KnowledgeBase results as cited context snippets, not invisible hidden facts.

# Citations

Citations should include:

- source title
- path or URL
- section heading
- chunk id
- timestamp

# Worker Use

Workers should be instructed to cite KnowledgeBase artifacts when using them in outputs.
