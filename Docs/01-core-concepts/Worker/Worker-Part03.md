# Worker Specification (Part 3)

## Context
Workers receive only required context.

## Prompt
Generated by orchestrators.

## Permissions
Explicit permissions only.

## Sandboxes
Each worker runs in an isolated execution environment.

## Terminal
Each worker owns one terminal session.
